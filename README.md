# TopUser

TopUser is the combined userbot. It includes the personal userbot commands and the embedded telegram-terminal.

Install:

```bash
python3 -m venv remoteenv
source remoteenv/bin/activate
pip install -r requirements.txt
```

Launch from the repo root:

```bash
TG_API_ID=123456 TG_API_HASH=your_api_hash python3 personal-userbot.py
```

Details are in [TopUser/README.md](TopUser/README.md).
